Memory Game

The Memory Game was built to test a user's memory skills. The aim of the game is to match 8 sets of cards in pairs of two in a deck. The game is won once all the cards in the deck have been matched.

This project was built as a Udacity course project to demonstrate mastery of HTML, CSS, and JavaScript.
